# Controls

### Dolphin Emulator

This setup is by default using the player 1 wired pro controller (Device: SDL/0/Nintendo Switch Pro Controller).

If you are playing with multiple players, reassign the device when loading the profiles: 
- Player 1: ``SDL/0``
- Player 2: ``SDL/1``
- Player 3: ``SDL/2``
- Player 4: ``SDL/3``
